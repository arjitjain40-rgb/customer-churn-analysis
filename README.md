# Customer Churn Analysis Dashboard

## Overview
This project analyzes customer churn behavior using SQL and Power BI.

## Key Insights
- Overall churn rate is 26.54%
- Month-to-month contracts show highest churn
- New customers (0–1 year) are most likely to churn
- High monthly charges correlate with higher churn

## Tools Used
- SQL (MySQL)
- Power BI

## Features
- Interactive dashboard with filters
- KPI cards for churn metrics
- Visual analysis by contract, tenure, and charges

## Dataset
Telco Customer Churn Dataset
